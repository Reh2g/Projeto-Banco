# Projeto-Banco
Meu primeiro projeto em Python da Faculdade!

● Projeto da matéria CC1612 - FUNDAMENTOS DE ALGORITMOS, primeiro semestre (Individual).

Objetivo - Criar um sistema de banco em Python com as seguintes ações:
Criar conta;
Apagar conta;
Debitar;
Depositar;
Mostrar saldo;
Mostrar extrato.
